Installation Guide

Follow these simple steps to set up the plugin:

  1. Download the plugin file from our website.
  2. Open IntelliJ and navigate to **Settings > Plugins**.
  3. Click on **Install Plugin from Disk** and select the downloaded file.
  4. Restart IntelliJ and start using the plugin.
